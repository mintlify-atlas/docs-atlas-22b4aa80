## Welcome to E-commerce Backend API

A modern, production-ready e-commerce backend built with Spring Boot 3.5 that provides comprehensive REST APIs for managing products, shopping carts, orders, and user authentication. The system leverages event-driven architecture with Kafka for seamless microservices integration.

## Technology Stack

- **Spring Boot 3.5.4** - Modern Java framework
- **Spring Security** - Authentication and authorization
- **Spring Data JPA** - Database access with Hibernate
- **MySQL** - Primary database
- **Keycloak** - Identity and access management
- **Apache Kafka** - Event streaming platform
- **Docker** - Containerization support
- **OpenAPI/Swagger** - API documentation

## Architecture Overview

The system follows a microservices architecture with event-driven communication:

1. **REST API Layer** - Spring Boot controllers handle HTTP requests
2. **Service Layer** - Business logic and transaction management
3. **Data Layer** - JPA repositories and MySQL database
4. **Event Layer** - Kafka producers and consumers for inter-service communication
5. **Security Layer** - JWT validation and Keycloak integration

<Note>
This API is designed to work as part of a larger microservices ecosystem, communicating with inventory, core, and other services via Kafka events.
</Note>

## Core Capabilities

### User Management
- User registration with email verification
- JWT-based authentication
- Profile management with addresses
- Password reset functionality

### Product Catalog
- Paginated product listings
- Advanced filtering by brand, category, and price
- Product reviews and ratings
- Favorites management
- Related products recommendations

### Shopping Cart
- Add, update, and remove items
- Persistent cart storage
- Real-time stock validation

### Order Processing
- Purchase creation and tracking
- Order status management
- Event-driven inventory updates
- Purchase history
