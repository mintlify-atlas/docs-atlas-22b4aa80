---
title: "Get Purchase by ID"
api: "GET /purchase/{id}"
description: "Retrieve a specific purchase with full cart and product details"
---

## Authentication

This endpoint requires Bearer token authentication. The user must have an active session.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the purchase
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

<ResponseField name="id" type="integer">
  Purchase ID
</ResponseField>

<ResponseField name="date" type="string">
  Purchase date (ISO 8601 format)
</ResponseField>

<ResponseField name="reservationTime" type="string">
  Reservation timestamp (ISO 8601 format)
</ResponseField>

<ResponseField name="direction" type="string">
  Delivery address
</ResponseField>

<ResponseField name="status" type="string">
  Purchase status (PENDING, CONFIRMED, or CANCELLED)
</ResponseField>

<ResponseField name="cart" type="object">
  Cart details
  <ResponseField name="id" type="integer">
    Cart ID
  </ResponseField>
  <ResponseField name="finalPrice" type="float">
    Total cart price
  </ResponseField>
  <ResponseField name="items" type="array">
    Array of cart items
    <ResponseField name="id" type="integer">
      Cart item ID
    </ResponseField>
    <ResponseField name="quantity" type="integer">
      Product quantity
    </ResponseField>
    <ResponseField name="product" type="object">
      Product information
      <ResponseField name="id" type="integer">
        Product ID
      </ResponseField>
      <ResponseField name="title" type="string">
        Product name
      </ResponseField>
      <ResponseField name="description" type="string">
        Product description
      </ResponseField>
      <ResponseField name="price" type="float">
        Unit price
      </ResponseField>
      <ResponseField name="mediaSrc" type="array">
        Array of product image URLs
      </ResponseField>
      <ResponseField name="productCode" type="integer">
        Product code identifier
      </ResponseField>
    </ResponseField>
  </ResponseField>
</ResponseField>

## Error Responses

- **401 Unauthorized**: User is not authenticated or session is inactive
- **200 OK with null body**: Purchase with specified ID does not exist

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/purchase/42" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/purchase/42', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const purchase = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/purchase/42',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)
purchase = response.json()
```
</CodeGroup>

## Related Endpoints

- **GET /purchase/pending-cart**: Get the user's last pending purchase within 4 hours
- **POST /purchase/{id}/confirm/{addressId}**: Confirm a pending purchase
- **DELETE /purchase/{id}**: Cancel a purchase and restore stock

## Notes

- Returns full cart details including all items and products
- Returns 200 OK with null body if purchase doesn't exist (not 404)
- Use this endpoint to display order details and confirmation pages
