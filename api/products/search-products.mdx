## Endpoint

```
GET /products/search
```

## Description

Searches for products by title using a text query with support for synonyms. Only active products are returned. The search is case-insensitive and matches partial titles.

## Authentication

No authentication required.

## Query Parameters

<ParamField query="query" type="string" required>
  Search query to match against product titles. Supports synonym expansion based on configured synonyms.
</ParamField>

## Search Behavior

### Case Insensitivity
All searches are case-insensitive (ProductSearchController.java:45)
```java
String lowerQuery = query.toLowerCase();
```

### Synonym Expansion
The search supports synonym expansion using a configured synonyms.json file:
- If the query matches a key in the synonyms map, related types are included (ProductSearchController.java:47-49)
- Example: searching for "laptop" might also match "notebook" or "computer"
- Synonyms are loaded from `src/main/resources/synonyms.json` (ProductSearchController.java:35)

### Matching Logic
Products match if (ProductSearchController.java:52-55):
- Product has `active=true`
- Product has a non-null title
- Title contains the search query OR any synonym terms

### Full-Text Search
The search uses substring matching:
```java
p.getTitle().toLowerCase().contains(lowerQuery)
```

## Implementation Notes

- Returns full Product entities, not DTOs (ProductSearchController.java:44)
- No pagination - returns all matching results
- Synonyms file is optional - if not found, only direct matches are returned
- Products with null titles are excluded from results
- Search is performed in-memory after fetching all products

## Synonym Configuration

Synonyms are configured in `src/main/resources/synonyms.json`:
```json
{
  "laptop": ["notebook", "computer", "pc"],
  "phone": ["smartphone", "mobile", "cellphone"],
  "headphones": ["earphones", "headset", "earbuds"]
}
```

When a user searches for a key term, the system also searches for products containing any of the synonym values in their titles.
