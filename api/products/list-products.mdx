## Endpoint

```
GET /products
```

## Description

Retrieves all active products from the catalog with pagination support. Only products with `active=true` are returned.

## Authentication

No authentication required.

## Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number for pagination (0-based)
</ParamField>

<ParamField query="size" type="integer" default="20">
  Number of items per page
</ParamField>

<ParamField query="sort" type="string">
  Sorting criteria (e.g., `price,asc` or `title,desc`)
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/products?page=0&size=10" \
  -H "Content-Type: application/json"

const response = await fetch('https://api.example.com/products?page=0&size=10', {
  method: 'GET',
  headers: {

  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/products',
    params={'page': 0, 'size': 10}
)

data = response.json()
```

## Implementation Notes

- Only products with `active=true` are returned (line 64 in ProductController.java:64)
- Results are filtered after pagination to show only active products
- Default page size is 20 items
- Supports standard Spring Data pagination parameters
