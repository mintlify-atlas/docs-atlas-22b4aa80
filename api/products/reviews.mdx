## Add Review

### Endpoint

```
POST /products/code/{productCode}/review
```

### Description

Adds a review (rating and comment) for a product. Updates the product's average calification and emits an event to the Core system.

### Authentication

**Required**: JWT Bearer token

Authenticated users can only submit one review per product.

### Path Parameters

<ParamField path="productCode" type="integer" required>
  The product code (external identifier) of the product to review
</ParamField>

### Request Body

<ParamField body="calification" type="float" required>
  Rating value for the product (typically 1-5)
</ParamField>

<ParamField body="description" type="string">
  Text review/comment for the product
</ParamField>

### Request Schema

<Expandable title="ReviewRequest">
  ```java
  {
    "calification": float,
    "description": string
  }
  ```
</Expandable>

### Response

<ResponseField name="productId" type="integer">
  Product ID
</ResponseField>

<ResponseField name="productTitle" type="string">
  Product title
</ResponseField>

<ResponseField name="promedio" type="float">
  Updated average rating for the product (calculated from all reviews)
</ResponseField>

<ResponseField name="reviews" type="array">
  Array of all reviews for this product
  
  <Expandable title="ReviewDTO">
    <ResponseField name="id" type="long">
      Review unique identifier
    </ResponseField>
    
    <ResponseField name="calification" type="float">
      Rating value
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Review text
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```bash cURL

  -H "Content-Type: application/json" \

    "calification": 4.5,
```

## Implementation Notes

### Review Creation
- Each user can only submit one review per product (ProductController.java:157-160)
- Product's `calification` field is automatically updated with the new average (ProductController.java:171-172)
- Review creation emits an event to the Core system via ECommerceEventService (ProductController.java:174-180)
- Anonymous reviews are allowed - user is optional (ProductController.java:164)

### Average Calculation
The average rating (promedio) is calculated as:
```java
float promedio = (float) reviews.stream()
    .mapToDouble(Review::getCalification)
    .average()
    .orElse(0.0);
```
Returns 0.0 if no reviews exist (ProductController.java:169, 210)
