## Endpoint

```
GET /products/{id}
```

## Description

Retrieves a single product by its unique identifier along with related products based on shared brand or categories. Only active products are returned.

## Authentication

No authentication required.

## Path Parameters

<ParamField path="id" type="long" required>
  The unique identifier of the product to retrieve
</ParamField>

## Response

Returns a ProductWithRelatedDTO object containing the main product and related products.

## Implementation Notes

- Product must have `active=true` to be returned (ProductController.java:290)
- Related products are determined by:
  - Same brand ID (ProductController.java:297)
  - OR shared category IDs (ProductController.java:298)
- Related products exclude the requested product itself (ProductController.java:296)
- Only active related products are included (ProductController.java:295)
- Related products are not paginated
