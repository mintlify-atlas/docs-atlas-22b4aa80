---
title: "Remove Item from Cart"
api: "DELETE /cart/{id}"
description: "Delete a cart and all its items"
---

## Authentication

This endpoint requires Bearer token authentication. The user must have an active session.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the cart to delete
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

Returns a 200 OK status with no body on successful deletion.

## Error Responses

- **401 Unauthorized**: User session is not active or invalid token
- **404 Not Found**: Cart with the specified ID does not exist

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.example.com/cart/1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/cart/1', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

if (response.ok) {
  console.log('Cart deleted successfully');
}
```

```python Python
import requests

response = requests.delete(
    'https://api.example.com/cart/1',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

if response.status_code == 200:
    print('Cart deleted successfully')
```
</CodeGroup>

## Notes

- Deleting a cart will also remove all associated cart items due to cascade delete
- This operation is permanent and cannot be undone
- Ensure the cart is no longer needed before deletion
- To cancel a purchase and restore stock, use the purchase cancellation endpoint instead
