---
title: "Clear Cart"
api: "DELETE /cart/{id}"
description: "Clear all items from a cart by deleting the cart entirely"
---

## Overview

This endpoint deletes the entire cart including all its items. The cart system in this API uses a delete operation to clear carts.

## Authentication

This endpoint requires Bearer token authentication. The user must have an active session.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the cart to clear
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

Returns a 200 OK status with no body when the cart is successfully cleared.

## Error Responses

- **401 Unauthorized**: User session is not active or invalid token
- **404 Not Found**: Cart with the specified ID does not exist

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.example.com/cart/1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/cart/1', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

if (response.ok) {
  console.log('Cart cleared successfully');
}
```

```python Python
import requests

response = requests.delete(
    'https://api.example.com/cart/1',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

if response.status_code == 200:
    print('Cart cleared successfully')
```
</CodeGroup>

## Notes

- This operation removes the cart and all associated items
- The operation is permanent and cannot be undone
- All cart items are deleted through cascade deletion
- To cancel a purchase and restore product stock, use the purchase cancellation endpoint instead
