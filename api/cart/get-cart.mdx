---
title: "Get Cart by ID"
api: "GET /cart/{id}"
description: "Retrieve a specific cart by its ID"
---

## Authentication

This endpoint requires Bearer token authentication. The user must have an active session.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the cart
</ParamField>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The unique identifier of the cart
</ResponseField>

<ResponseField name="finalPrice" type="float">
  The total price of all items in the cart
</ResponseField>

<ResponseField name="items" type="array">
  List of cart items
  <ResponseField name="id" type="integer">
    The cart item ID
  </ResponseField>
  <ResponseField name="quantity" type="integer">
    The quantity of this product in the cart
  </ResponseField>
  <ResponseField name="product" type="object">
    Product details
    <ResponseField name="id" type="integer">
      Product ID
    </ResponseField>
    <ResponseField name="title" type="string">
      Product title
    </ResponseField>
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    <ResponseField name="price" type="float">
      Product unit price
    </ResponseField>
    <ResponseField name="stock" type="integer">
      Available stock quantity
    </ResponseField>
    <ResponseField name="mediaSrc" type="array">
      Array of image URLs
    </ResponseField>
    <ResponseField name="productCode" type="integer">
      Product code identifier
    </ResponseField>
  </ResponseField>
</ResponseField>

## Error Responses

- **401 Unauthorized**: User session is not active or invalid token
- **404 Not Found**: Cart with the specified ID does not exist

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/cart/1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/cart/1', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const cart = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/cart/1',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)
cart = response.json()
```
</CodeGroup>
