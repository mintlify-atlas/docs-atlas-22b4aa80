## Get All Brands

<api method="GET" endpoint="/brands" />

Retrieve all brands in the system.

### Response

Returns an array of BrandDTO objects:

<ResponseField name="brands" type="array">
  <ResponseField name="id" type="long">
    Brand unique identifier
  </ResponseField>
  <ResponseField name="name" type="string">
    Brand name
  </ResponseField>
  <ResponseField name="active" type="boolean">
    Whether the brand is active
  </ResponseField>
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.example.com/brands"
```

```javascript JavaScript
fetch('https://api.example.com/brands')
  .then(response => response.json())
  .then(data => console.log(data));
```

```python Python
import requests

response = requests.get('https://api.example.com/brands')
brands = response.json()
```

</CodeGroup>

### Example Response

```json
[
  {
    "id": 1,
    "name": "Nike",
    "active": true
  },
  {
    "id": 2,
    "name": "Adidas",
    "active": true
  },
```

## Add Brand

<api method="POST" endpoint="/brands/add" />

Create a new brand in the system.

### Request Body

<ParamField body="name" type="string" required>
  Brand name
</ParamField>

<ParamField body="active" type="boolean">
  Whether the brand should be active (defaults to true)
</ParamField>

### Response

Returns the created BrandDTO object:

<ResponseField name="id" type="long">
  Brand unique identifier
</ResponseField>

<ResponseField name="name" type="string">
  Brand name
</ResponseField>

<ResponseField name="active" type="boolean">
  Whether the brand is active
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl -X POST "https://api.example.com/brands/add" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "New Balance",
    "active": true
  }'
```

```javascript JavaScript
fetch('https://api.example.com/brands/add', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'New Balance',
    active: true
  })
})
.then(response => response.json())
.then(data => console.log(data));
```

```python Python
import requests

data = {
    'name': 'New Balance',
    'active': True
}

response = requests.post('https://api.example.com/brands/add', json=data)
brand = response.json()
```

</CodeGroup>

### Example Response

```json
{
  "id": 4,
  "name": "New Balance",
  "active": true
}
```

### Notes

- If a brand with the same name already exists (case-insensitive), the existing brand is returned
- The `active` field defaults to `true` if not provided

## Brand Entity Structure

The Brand entity contains the following fields:

- **id** (Integer): Auto-generated unique identifier
- **name** (String): Brand name (nullable)
- **active** (boolean): Active status (defaults to true)
- **brandCode** (Integer): Stable code used across microservices (unique, nullable)
- **products** (List): Associated products (one-to-many relationship)

## Notes

- Mock synchronization endpoints (`/sync`, `/mock/*`) are disabled and return 501 Not Implemented
- Brand synchronization should be handled through the Communication API (Core)
- Brand deletion is a soft delete - brands are marked as inactive rather than removed
- Brand codes provide stable identifiers across microservices
